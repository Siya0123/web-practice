WHAT IS HTML?

  • HTML stands for Hyper Text Markup Language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the content
  • HTML elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.

WHAT IS CSS?

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in other media
  • CSS saves a lot of work. It can control the layout of multiple web pages all at once
  • External stylesheets are stored in CSS files

TIPS AND TRICKS TO LEARN WEB DEVELOPMENT EASILY

1. Focus on HTML and CSS in starting

Sharing my own experience, the main thing I did wrong was I started all 3 languages together and thats a wrong way to learn web developing If you wanna become a back-end developer then you can focus on javascript more than HTML and CSS but if you wanna become a front-end or full stack web developer then you should firstly focus on HTML and CSS. You can learn javascript later but you can learn basic javascript not intemediate or advance.

2. Keep learning Keep making

Keep learning Keep making is a fast and easy way to learn web development. Keep learning and Keep making means learn around 5-6 topic of HTML and CSS daily and add those topics to your practice projects then you can understand in a better way and you can remember the codes easily.

3. Learn from theory not videos

Learn from theory not videos is a effective way to learn code because if you learn your codes from videos so the teacher will teach you in his/her own way. But if you will learn the code from theory like from w3schools and tutorialspoint so you will learn the codes in your own way and you can also get the source code of every topics easily and anytime.

4. Revise topics

Revision of topics plays a important role in learning any language if you are learning codes about 2 hours a day so it will basically take 30-40 minute to revise all topics and don't revise by reading the topics code that topic in your code edtior. So you can easily remember the codes for next day.

5. Built your projects with a new programmer

If you have a code buddy to built a projects so you can learn smart way to built a page. Because both of you will share your ideas to each other and create awesome pages. Also you can gain some experience in coding and you can prepare yourself for team work projects.
